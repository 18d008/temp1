Githubページ

Githubページを使うと誰でも無料で簡単にホームページを立ち上げることができます.

レスポンシブデザイン

このサンプルホームページはレスポンシブデザインで作られています. パソコンのような大きなディスプレイで見てもスマホのような小さなディスプレイで見ても縦長画面でも横長画面で見てもサイズにあわせてよしなに見た目を変えてくれます.

ホームページの応用

今後演習を通して「データベース」「ログイン」「API」を追加して簡単なアプリを作っていきます.

ログイン

個人情報を扱うアプリにはログインの機能を追加して自分以外のユーザがアクセスできないようにセキュリティを強化します.

データベース

データを整理整頓してわかりやすく表示・操作することがITシステムの主な機能でありデータベースがそれを担います.

API

APIはクラウド側で人工知能のような高度な処理をアプリから実行するためのインタフェースを表します.
